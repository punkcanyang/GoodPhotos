# GoodPhoto v0.1.0 - Alpha Release 🎉

Welcome to the first public beta of **GoodPhoto (优选照片)**! 
这个版本标志着一个“AI 原生”摄影与视觉设计审核工作台的诞生。只需连接您偏好的大模型（VLM）API，即可让您的本地电脑化身为一位顶级艺术指导与修图选片师。

---

## 🌟 核心亮点 (Highlights)

### 1. 广谱的多模态能力对接 (11 Providers Supported)
本版本底层重构了请求协议桥梁，现已完美接入当前地表最强的 11 种大语言视觉模型：
- **顶级旗舰体验**：无缝对接 `GPT-4o/5.4` 与 `Google Gemini 1.5/2.0`。
- **国产性价比之光**：深度集成 **DeepSeek-VL2**、**字节豆包 (Doubao Vision Pro)**、**智谱 GLM-4V** 与 **通义千问 (Qwen-VL-Max)**。
- 无论你是海外用户还是国内开发者，总有一个网络通畅且极具性价比的高智商模型适合你的工作流。

### 2. 九大专业审判流派 (9 Professional Aesthetics Profiles)
告别一句干瘪的“这张图好看吗？”。我们为 AI 加载了各行精尖的“系统级偏见”：
- **顶级画廊纪实**：以 *Magnum Photos* 与顶级摄影展的叙事结构要求进行排雷与打分。
- **网感与情绪选片**：专攻小红书 / Instagram 高光肤色爆款的快速捞取。
- **UI/UX & 排版专家**：以 *Apple HIG* 的极客眼光找出设计稿和海报中对齐与层级的致命缺陷。
- **跨界视觉**：支持概念原画审查、空间室内渲染材质 PBR 评分，以及支持 OCR 原生结构化名片和产品物料的信息提取！
- **合影防眨眼杀手**：全自动扫描排查多人连拍合影，一票淘汰闭眼半闭眼素材。

### 3. 一切都为了最终的“可用性”得分
不再单纯写漫长的赞美词，大模型现在会在后台为你隐性制定 **8个要素** 标准，并雷打不动地从以下 6 个严酷维度进行统分：
`构图张力` · `光影层次` · `情绪强度` · `叙事含量` · `视觉记忆点` · `冗余度(扣分)`

### 4. Stability AI 无痕除杂 (Inpainting)
内置了针对冗余要素的本地放大“草图涂鸦”能力。看到穿帮路人或多余水杯，直接在软件内涂黑即可调用 *Stability AI Erase* 直接将其在原图画幅内剔除。

### 5. 专业后期闭环 (Adobe Bridge)
你辛辛苦苦等大模型跑出的评估分数不会只是一个看看就没用的数字。应用支持一键在此图所在目录生成标准的 `.xmp` Sidecar 侧边信息文档；随后你将原图拖入 **Lightroom** 或 **Capture One** 时，原本电脑里的“高分推荐/低分废片”会自动对应为红黄绿等色标和星级。

## 💻 下载与安装 (Installation)

由于本应用建立在极为细致的本地底层文件级操作 (Tauri v2 + Rust) 上，此版本为所有 macOS (Apple Silicon M1/M2/M3) 用户提供了编译好的直接运行包。

- **`GoodPhotos_0.1.0_aarch64.dmg`**: 适合日常使用的镜像安装包。（下载后双击挂载，将应用拖拽入 `Applications` 文件夹即可）
- *注：初次在 Mac 上运行可能提示来自未认证开发者（目前为开源 Alpha 免签版本），请进入 `系统设置 -> 隐私与安全性 -> 仍然打开` 放行。*

## 🛠️ 初始化注意点
1. 本程序**不含内置订阅点数**，属于 BYOK *(Bring Your Own Key)* 架构。
2. 首次运行请务必点击左上角齿轮⚙️，在任意一个主流模型（比如 Qwen 或 DeepSeek）一栏填入你自己申请好的 API Key，否则将无法激活 AI 阅片面板。
